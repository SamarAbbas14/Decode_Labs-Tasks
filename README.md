# Python Mini Projects

This repository contains four beginner-friendly Python projects that helped me practice core programming concepts such as loops, conditionals, lists, exception handling, dictionaries, and working with Python's built-in libraries.

These projects are simple but practical and were created to strengthen my understanding of Python fundamentals.

---

## Projects Included

### 1. To-Do List

A simple command-line application for managing daily tasks.

**Features**
- Add new tasks
- Delete existing tasks
- View all saved tasks
- Exit the program safely

**Concepts Used**
- Lists
- Loops
- Conditional statements
- User input
- String comparison

---

### 2. Expense Tracker

A basic expense calculator that keeps adding expenses until the user decides to stop.

**Features**
- Add multiple expenses
- Prevent negative values
- Handle invalid inputs
- Display the final total expense

**Concepts Used**
- Exception handling (`try` / `except`)
- Loops
- Variables
- Input validation

---

### 3. Password Generator

A secure password generator that creates strong random passwords using Python's built-in libraries.

**Features**
- User chooses password length
- Minimum length of 15 characters
- Maximum length of 64 characters
- Generates a password using letters, numbers, and symbols

**Libraries Used**
- `secrets`
- `string`
- `time`

---

### 4. General Knowledge Quiz

A small quiz game that asks multiple questions and calculates the user's score.

**Features**
- Multiple questions
- Instant feedback
- Final score display

**Concepts Used**
- Lists
- Dictionaries
- Loops
- String methods
